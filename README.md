# OpenPad

A minimal, shareable notepad that lives entirely in the URL. Write a note, hit copy — the entire note is encoded into a short link. No accounts, no database, no backend.

## Features

- Rich text editing (bold, italic, code)
- Notes stored in the URL — sharing a link shares the note
- Auto-save every 2 minutes
- Short links via TinyURL
- Works offline (PWA with service worker)
- Installable on mobile and desktop

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+B` | Bold |
| `Cmd+I` | Italic |
| `Cmd+J` | Code |
| `Cmd+S` | Save |
| `Cmd+C` | Copy link |
| `Cmd+X` | Clear |
| `Cmd+/` | Show shortcuts |
